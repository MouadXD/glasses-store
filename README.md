<h3>Run Locally</h3>
<h4>1. Install Dependencies</h4>
    
    npm install
    
    npm install react-router-dom@6
    
    npm install react-icons --save
    
    npm install react-redux @reduxjs/toolkit
    
<h4>2. Run development server</h4>
    
    npm run dev
